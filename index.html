Defining HTML and CSS Terms

  • Display - Changes how a HTML element behaves whether it's text or an image
  • Float - Positions HTML elements after it a certain way ( left or right ), similar to 'clustering'
  • Overflow - Changes how the overflowing content of a parent element behave whether hidden or scrollable or just allowed to overflow.
  • Ul - A type of lists that consists of default bullet points per list items ' li '
  • Li - A list item used in ul and ol tags, can come with bullet points, numbers, numerals ect..
  • Href - Stands for hypertext reference and used in links to do a certain action like taking you to a different page

Notes for lists: Each list must have a title Each list must have properties styled differently. (Different text colors, etc.) Review:
