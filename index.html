Defining HTML and CSS Terms

  • Display - Changes how a HTML element behaves whether it's text or an image
  • Float - Positions HTML elements after it a certain way ( left or right ), similar to 'clustering'
  • Overflow - Changes how the overflowing content of a parent element behave whether hidden or scrollable or just allowed to overflow.
  • Ul - A type of lists that consists of default bullet points per list items ' li '
  • Li - A list item used in ul and ol tags, can come with bullet points, numbers, numerals ect..
  • Href - Stands for hypertext reference and used in links to do a certain action like taking you to a different page

RUBRIC :

Lists: Create an unordered list of your favorite TV shows. (minimum of 3). Create another unordered list of your favorite movies using a different list-style-type. (minimum of 3). Create an ordered list of your favorite books. (minimum of 3). Create another ordered list of your favorite video games of a different list-style-type. (minimum of 3). Notes for lists: Each list must have a title Each list must have properties styled differently. (Different text colors, etc.) Review:

  • Use the HTML <ul> to define an unordered list like this list!
  • Use the HTML <ol> to define an ordered list like the following list after this!
  • Use the HTML <li> to define a list item like each bullet point / number line!
Tables: Create a table that includes a title with a minimum of 3 columns and 4 rows. Write anything you want of your interest in your table. Notes for tables: The table must have styling which must include a border, and at least 1 property. Review:
  1. Use the HTML <table> to define a table
  2. Use the HTML <tr> to define a table ROW
  3. Use the HTML <td> to define a table DATA
  4. Use the HTML <th> to define a table HEADING
  5. Use the HTML <caption> to define a table CAPTION
